[0:20 — WHAT IT IS]
EcoSort is an AI-powered municipal waste sorting agent built for the "Agents for Good" track. It leverages the Google Agent Development Kit (ADK) 2.0
 to accurately identify materials and securely fetch live, hyper-local recycling guidelines so you never contaminate a recycling bin again.

[1:00 — ARCHITECTURE]
(show assets/architecture_diagram.png)
Here's how the workflow is constructed. The user's input hits the Security Checkpoint first, which strips PII like phone numbers and
 detects prompt injections or hazardous medical waste. If it passes, the input moves to the Orchestrator Agent. The Orchestrator delegates
  identification to the Waste Analyzer Sub-Agent. Then, it uses the Local Guideline Sub-Agent, which connects directly to a local MCP Server
   to pull municipal rules. If the user's initial input is too vague, the flow routes to a Human Pause Node where it asks for clarification
    before proceeding.

[3:00 — SECURITY & MCP]
This strict security node ensures that prompt injections are blocked and PII is scrubbed before any tokens are sent to Gemini. Additionally, 
the MCP Server makes the Local Guideline agent entirely modular—cities can plug in their own database tools without us ever touching the agent code.
